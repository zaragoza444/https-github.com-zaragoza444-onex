Z ECOSYSTEM ICONS — FOR YOUR PC (WINDOWS)
=========================================

1. Download / unzip: Z-Ecosystem-Icons-PC.zip
2. Open folder: Desktop-Icons\
3. You will see:
   - Z-Bank.ico
   - Z-Blockchain.ico
   - Z-Payments.ico
   - Z-Wallet.ico
   - Z-Ecosystem.ico

SET SHORTCUT ICON
-----------------
1. Right-click any desktop shortcut
2. Properties → Change Icon…
3. Browse to Desktop-Icons\Z-Bank.ico (or other)
4. OK → Apply

COPY TO DESKTOP
---------------
Copy the whole Desktop-Icons folder to your Desktop for quick access.
